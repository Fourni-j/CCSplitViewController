# CCSplitViewController
Custom UISplitViewController like Facebook iPad application

## Installation

Use [CocoaPods](http://cocoapods.org)

```sh
pod 'CCSplitViewController', :git => 'https://github.com/Fourni-j/CCSplitViewController.git'
```
